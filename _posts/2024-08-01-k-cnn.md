> Some thoughs on developing code in K.

I've recently started taking a look at the ngn implementation of the k programming language.

Here is my attempt at creating a convolutional neural network in k, based upon the APL implementation of the same found [here](https://github.com/ashinkarov/cnn-in-apl/blob/master/cnn.apl)

```k
avg: { +/:/:[0] x}
maxpos: { (where x) ? max x}
conv: {a:{[a] b} 1 +/: count a -/: count b; (+/ a) */:/: (1 +/: count b) @/: a}
```
