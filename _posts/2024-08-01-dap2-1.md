> Going from Dap, to Dap2.

The purpose of this series is to teach statistics, by designing a programming language that is based on SAS. 


I am using ANTLR4, with Xueqilsj's SAS Grammar as a starting point (which can be found [here](https://github.com/xueqilsj/sas-grammar)).

So as an example, let's take the following code:

```sas
data test;
input x y z;
cards;
1 2 3
4 5 6
7 . 9
;
run;

proc print test;
run;
```
